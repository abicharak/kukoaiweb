// System Overview

Kuko AI is an experimental desktop agent developed to bridge the gap between voice commands and operating system actions. Built by students at ABSS Institute of Technology.

It leverages the Google Speech Recognition engine to execute local Windows tasks, manage files, and integrate with LLMs for intelligent responses.
